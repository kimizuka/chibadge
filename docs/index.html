chibadge（チバッジ）は千葉ロッテマリーンズのスコアをバッジで伝えてくれる世界一ミニマルな野球速報アプリでした。

いつでもどこでもマリーンズのスコアを確認する手段として開発しました。

2つのアプリのバッジにマリーンズの現在の得点 / 失点が表示されるため、アプリを開くことなくホーム画面上でマリーンズの試合のスコアが確認できました。

アップデート申請がなかなか通過せず、現在はストアから取り下げております。

chibadge shiro

chibadge shiro （チバッジ シロ）はバッジをつかってマリーンズの得点を伝えてくれるアプリでした。

単独ではマリーンズの勝敗は判りませんが、 大きな数字が表示されていると マリーンズファンをほんのりと嬉しい気持ちにしてくれました。

マリーンズが無得点のときはバッジがつきません。

アップデート申請がなかなか通過せず、現在はストアから取り下げております。

VS icon

VSアイコンは試合感をほんのりと演出します。

このページをホーム画面に追加するとVSアイコンになります。

chibadge kuro

chibadge kuro （チバッジ クロ）はバッジをつかってマリーンズの失点を伝えてくれるアプリでした。

単独ではマリーンズの勝敗は判りませんが、 大きな数字が表示されていると マリーンズファンをほんのりと哀しい気持ちにしてしまいました。

マリーンズが無失点のときはバッジがつきません。

アップデート申請がなかなか通過せず、現在はストアから取り下げております。

how to use

chibadge shiro・VSアイコン・chibadge kuro の3つをホーム画面に並べるだけでいつでもどこでも千葉ロッテマリーンズのスコアを確認できてました。

プッシュ通知の許可が必要でした。
